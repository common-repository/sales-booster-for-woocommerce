Sales Booster for WooCommerce shows the latest sales info on your site using toast notifications. It increases sales with effective communications.

== Description ==
Sales Booster for WooCommerce shows a toast notification to the visitors about the latest sold product mentioning how long ago the product was sold. It gives the visitors an idea about the trending products as well. This plugin comes with two different types of toast notifications. The first one shows the latest sold product with sale-time regardless of pages people visit. The second type of toast notification shows the latest sale info of any product that you visit. In case of opening an unsold product page, it shows the sale info of any product that has been sold lately. Moreover, an admin can customize the notification placements, transition effects, duration, behavior and so on.


= Features =
  * Impressive Host notification design
  * Can be show in Top left, Top right, Bottom left, Bottom right
  * Show hide animation
  * Product single page show current product’s notification if there any sales. If current product have no sale, then it will show a last sold product.
  * Product link in toast notification
  * User can close immediately
  * You can deactivate Sales Booster for WooCommerce from settings.
  * Impressive UI
  * Show time can be set from settings, Default 5 sec

== Installation ==

1. Go to Dashboard > Plugins > Add New, then search Sales Booster for WooCommerce and click Install Now
2. Upload ‘sales-booster-for-woocommerce’ to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Click on the new menu item "Sales Booster for WooCommerce" and Setup your Settings.
